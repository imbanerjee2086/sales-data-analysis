# Sales Data Analysis

## 📌 Objective

Analyze sales data to derive business insights.

## 🛠️ Tools Used

* SQL Server

## 📊 Business Questions

* What is the total sales?
* Which region has highest sales?
* Who is the top customer?
* Which product performs best?

## 📂 Project Structure

* dataset/ → Raw data creation
* analysis/ → SQL queries

## 📈 Key Insights

* Laptop is the top-selling product
* East region shows strong performance
* Raj is one of the top customers

## 🚀 Learnings

* Aggregations (SUM, GROUP BY)
* Business-oriented SQL thinking
* Writing structured queries
